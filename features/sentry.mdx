### Follow documentations
Sentry is enabled by default in your project to send errors data to Sentry. You just need to updated `SENTRY_LARAVEL_DSN` value in your .env file. Follow this documentations to learn more

```https://docs.sentry.io/platforms/php/guides/laravel/```
