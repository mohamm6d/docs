Xeepp provides seamless management of all development configurations at your fingertips. Simply open the resources/js.config.js file and update the configuration variables as needed. 

```json
  "site_under_construction": false,
  "primary_color": "#000000",
  "primary_color_name": "gray",
  "primary_font": "Inter",
  "background_color": "#f0f0f0",
  "currency": "EUR",
  ...
```
