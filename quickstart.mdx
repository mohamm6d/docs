## Setup your local

### Fork the Repository
Visit the GitHub Repository:

Go to the Xeepp Laravel Starter Kit repository on GitHub.

- Fork the Repository:

In the upper right corner of the repository page, click on the Fork button. This action creates a copy of the repository under your GitHub account.

### Clone Your Forked Repository
Copy the Repository URL:

Navigate to your forked repository on GitHub. Click on the Code button and copy the URL provided. You can choose between HTTPS or SSH based on your preference.

### Clone the Repository Locally:

Open your terminal or command prompt and run the following command to clone the repository to your local machine:
```
git clone https://github.com/your-username/lara-starter.git```
Replace https://github.com/your-username/lara-starter.git with the URL you copied from your forked repository.

- Navigate to the Project Directory:

Change into the directory of your cloned project:
```
cd lara-starter```
